# QuizForge Web

Next.js frontend for QuizForge.

## Scripts

- `pnpm dev` starts Next.js on port 3000.
- `pnpm build` creates the production Next.js build.
- `pnpm start` serves the production build.
- `pnpm lint` runs ESLint.

## Environment

Set `NEXT_PUBLIC_API_BASE_URL` to the Express API base URL, for example:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:4000/api
```
